# React Native AI - FAL Provider Skills

Claude Code skills for `@umituz/react-native-ai-fal-provider` - FAL.ai integration.

## Installation

```bash
npx skills add /Users/umituz/Desktop/github/umituz/apps/mobile/npm-packages/react-native-ai-fal-provider/skills/SKILL.md -g
```

## Usage

Say: **"Setup FAL AI in my app"**

## Features

- Text-to-image generation (Flux models)
- Image-to-video generation
- Face swap
- Background removal
- Image upscaling
- Cost tracking

## Supported Models

- Flux Pro Ultra/Schnell
- Fast SVD (video)
- Face Swap
- Background Removal (Rembg)
- ESRGAN (upscaling)
